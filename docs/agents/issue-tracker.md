# Issue Tracker: GitHub

Issues for this repository are managed using GitHub Issues.

## Workflows

### Creating an Issue

- Tool: `gh issue create`
- Command: `gh issue create --title "<title>" --body "<body>" --label "<label>"`

### Listing Issues

- Tool: `gh issue list`
- Command: `gh issue list --label "<label>"`

### Reading an Issue

- Tool: `gh issue view`
- Command: `gh issue view <number> --json title,body,state,comments`
