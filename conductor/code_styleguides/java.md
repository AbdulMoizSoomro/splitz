# Java Code Style & Development Principles

## Test Driven Development (TDD)

- **CRITICAL: Always write tests before implementation.**
- Follow the Red-Green-Refactor cycle for all Java development.
- Target >80% code coverage for all new or modified Java classes.

## Java Skills & Standards

- **Mandatory Skill**: Utilize the `java-springboot` skill for all Java and Spring Boot tasks to ensure expert-level standards and idiomatic patterns.
- Adhere to Java 21 and Spring Boot 3.2 conventions as defined in the skill instructions.
- Maintain a clean separation of concerns across all layers (Controller, Service, Repository, DTO, Mapper).
