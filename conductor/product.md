# Initial Concept

A Splitwise-like expense splitting application for friends and roommates, built with microservices architecture using Spring Boot 3.2 and Java 21. It aims to simplify the process of tracking shared costs, managing debts, and settling balances among a group of people.

# Product Guide: Splitz

## Vision Statement

Splitz is the definitive microservices-powered platform for transparent and precise expense management. It empowers friends, roommates, and travel groups to effortlessly split costs with complex logic, providing deep visual insights into their shared financial lives.

## Target Audience

- **Friends & Roommates**: Managing shared household bills, rent, and common grocery costs.
- **Travel Groups**: Tracking trip-specific expenses, from flights and accommodation to shared meals and activities.

## Core Value Proposition

- **Complex Split Logic**: Beyond simple 50/50 splits, Splitz supports unequal amounts, percentages, and shares, ensuring that every expense is divided exactly as intended.
- **Microservices Robustness**: Built for scale and reliability using a modern Java/Spring Boot architecture.
- **Architectural Transparency**: The system's "as-built" state is fully documented with visual Mermaid.js diagrams, ensuring long-term maintainability and clear service boundaries.
- **Visual & Analytical Clarity**: Users aren't just tracking numbers; they're gaining insights through intuitive dashboards, detailed history, and spending analytics.

## Key Features & Goals

- **Precise Expense Tracking**: Create and manage expenses with flexible split configurations.
- **Advanced Split Logic**: Support for equal, exact, percentage, shares, and adjustment-based splitting.
- **Group Management**: Organize users into persistent groups for ongoing cost sharing.
- **Custom Category Management**: Create and manage expense categories with custom icons and colors for better organization.
- **Settlement Engine**: Track who owes whom and provide a clear path to settling debts.
- **Financial Analytics**: Visualize spending patterns and enriched group balances with user profile details.

## Success Metrics

- **Accuracy**: Zero discrepancies in balance calculations and settlement totals.
- **User Adoption**: Growing number of active users and expense entries.
- **System Performance**: Low latency in microservice communication and data retrieval for analytics.
